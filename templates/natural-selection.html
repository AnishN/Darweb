Natural Selection Simulation

(Click HERE to go back to the home page)

Description:

This simulation illustrates how allele frequencies change over time in response to natural selection on diploid genotypes. Users may set fitnesses from 0.0 to 1.0 for each of the three genotypes. The number of generations is fixed at 100. Results from different iterations are displayed simultaneously in different colors.

Instructions:

  • Set a starting allele frequency from the first list box (range: 0.0 to 1.0; default: p = 0.5).
  • Set a fitness of A11 (range: 0.0 to 1.0; default: w11 = 0.5).
  • Set a fitness of A12 (range: 0.0 to 1.0; default: w12 = 0.5).
  • Set a fitness of A22 (range: 0.0 to 1.0; default: w22 = 0.5).
  • Push "Start" to begin a simulation.
  • Push "Clear" to clear the accumulated results and start over.

How to intepret the graph:

  • The vertical axis shows allele frequency.
  • The horizontal axis shows generation number.
  • Each color corresponds to a different iteration.

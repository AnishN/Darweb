Genetic Drift Simulation

(Click HERE to go back to the home page)

Description:

This simulation illustrates how allele frequencies change over time as a result of genetic drift in small populations. Users may set starting allele frequencies (0.0 to 1.0), population sizes (10 to 250), and numbers of generations for the simulation (50 to 250). Results from different iterations are displayed simultaneously in different colors.

Instructions:

  • Set a starting allele frequency from the first list box (range: 0.0 to 1.0; default: p = 0.5).
  • Set a population size (range: 10 to 1000; default: N = 50).
  • Set a maximum number of generations (range: 50 to 2500; default: 100).
  • Push "Start" to begin a simulation.
  • Push "Clear" to clear the accumulated results and start over.

How to intepret the graph:

  • The vertical axis shows allele frequency.
  • The horizontal axis shows generation number.
  • Each color corresponds to a different iteration.

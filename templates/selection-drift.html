Natural Selection and Genetic Drift Simulation

Description:

This simulation illustrates the interaction between natural selection and genetic drift. Users may set starting allele frequencies (0.0 to 1.0), population sizes (10 to 1000), and numbers of generations for the simulation (50 to 2500). Only a single set of fitnesses representing selection for an initially rare allele are employed, specifically w11 = 1.0, w12 = 0.9, w22 = 0.8. Results from different iterations are displayed simultaneously in different colors.

Instructions:

  • Set a starting allele frequency from the first list box (range: 0.0 to 1.0; default: p = 0.01).
  • Set a population size (range: 10 to 1000; default: N = 50).
  • Set a maximum number of generations (range: 50 to 2500; default: 100).
  • Push "Start" to begin a simulation.
  • Push "Clear" to clear the accumulated results and start over.
  • REMEMBER: There is directional selection in favor of A1. The fitnesses are: w11 = 1.0, w12 = 0.9, and w22 = 0.8.

How to intepret the graph:

  • The vertical axis shows allele frequency.
  • The horizontal axis shows generation number.
  • Each color corresponds to a different iteration.
